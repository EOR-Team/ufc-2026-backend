# 001: TTS 方案选择

**日期**: 2026-04-25
**状态**: 已通过
**决策者**: n1ghts4kura

## 背景

树莓派 4B 4GB + Coral TPU 环境下需要选择本地 TTS 方案，要求：
- 本地离线运行
- 中文支持
- 内存 < 1GB
- 无 GPU（CPU only）

## 考虑的方案

| 方案 | 内存占用 | 中文支持 | 质量 | 兼容性 | 结论 |
|------|----------|----------|------|--------|------|
| Piper TTS | <1GB | ✅ 单一音色 | 中等 | 树莓派优化 | **选择** |
| espeak-ng | <50MB | ✅ 机械音 | 差 | 通用 | 备选 |
| EdgeTTS | - | ✅ | 高 | 需联网 | **放弃** |
| Coqui XTTS | ~4GB | ✅ | 高 | 需 GPU | **放弃** |

## 决策

**选择:** Piper TTS

## 理由

1. 唯一满足内存约束的神经网络 TTS（<1GB）
2. 中文支持（单一音色 huayan）
3. 专为树莓派优化，medium 模型 3-5x 实时
4. 部署简单（pip 或二进制）

## 放弃的替代方案

- **EdgeTTS**: 需要联网，不适合离线边缘部署
- **Coqui XTTS**: 约 4GB VRAM 需求，树莓派环境无法满足

## 后续行动

- [ ] 验证 Piper 中文音色质量是否满足需求
- [ ] 如不满足，考虑微调 Piper 模型
