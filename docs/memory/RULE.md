# Memory Bank 管理原则

## Part 1: Currently Working

**ATTENTION**: 进行任何操作之前，检查当前的时间，确保**时效性**。

当你想要修改这一部分的内容的时候，按照下面的工作流程进行修改：

0. 按照 `docs/RULE.md` 的约束 **更新 Metadata**

1. 修改 `工作整体方向` - 用 `名词+动词` 的组合方式起名

2. 修改 `当前具体任务` - 用 `动词+名词` 的形式描述任务；不允许同时有多个任务，如果你认为用一句话无法概括完整个任务，**请求用户拆分任务**

3. 修改 `任务的作用` - 用一句话概括目前正在工作的具体任务的作用/意义

4. 修改 `任务的工作目录` - 以 *列表* 的形式罗列出 *将要修改* / *已经修改* 的目录；如果在一个父级目录中有多个子目录为工作目录，那么将子目录按 `父目录/子目录` 的形式列出，而不是直接列出父目录

5. 修改 `任务开始时间` - 只允许在 *开始新任务* 的时候进行修改

6. 修改 `任务预期结束时间` - 使用 **AskUserQuestion Tool** 询问用户预期结束时间。如果用户选择 不知道 选项，那么你就自行估测

7. 修改 `任务状态` - `正在进行` / `已完成`
